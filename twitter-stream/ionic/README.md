# Twitter Stream: Ionic sample

## Install

```sh
  npm install -g cordova ionic
```

```sh
  ionic state restore
```

## Config

Edit  www/js/app.js and point apiBase to your URL.


## Usage

```sh
ionic serve
```

Will start a new tab in you browser pointing to [http://localhost:8100](http://localhost:8100)

## Mobile

To run on iOS:
  ionic run ios

To run on Android:
  ionic run android
